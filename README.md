CareerCarve API
================

This is the CareerCarve API, built using Node.js and Express.js.

Endpoints
----------

### Mentors

* `GET /mentors`: Fetch all mentors
* `GET /mentors/:id`: Fetch a mentor by ID

### Students

* `GET /students`: Fetch all students
* `GET /students/:id`: Fetch a student by ID
